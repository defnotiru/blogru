# blogru

guides:

https://developer.mozilla.org/en-US/

how 2 upload:

open terminal (ctrl+alt+t on linux)

navigate to blogru (cd blogru on linux)

stage changes (git add -A)

commit changes (git commit -am "updated")

push to servers (git push)

or run "iru-run-me-to-upload"

after pushing, your update will be available at blogruu.netlify.app in a minute or so
